    Constructors

    NameDescription
    client (hostname, port) Connect to resolved hostname and port. Uses the first resolved address the socket can successfully connect to.
    client (hostname, service) Connect to resolved hostname and service. Uses the first resolved address the socket can successfully connected to.
    client (addr) Connect to remote socket address.

    server (hostname, port) Create a UDP socket and bind it to the resolved hostname and port. Uses the first resolved address the socket can successfully bind. A port number of zero will be bound to an ephemeral port.
    server (hostname, service) Create a UDP socket and bind it to the resolved hostname and service. Uses the first resolved address the socket can successfully bind.

    server (addr, port) Create a UDP socket and bind it to IP addr and port. A port number of zero will be bound to an ephemeral port.

    Example

    auto server = UDP.server(IPv4Addr(127, 0, 0, 1));
auto client = UDP(ProtocolFamily.IPv4);

ubyte[4] ping = ['p', 'i', 'n', 'g'];
client.sendTo(server.localAddr, ping[]);

ubyte[4] buf;
auto res = server.recvFrom(buf[]);
assert(res.size == 4);
assert(buf[] == ping[]);

ubyte[4] pong = ['p', 'o', 'n', 'g'];
server.sendTo(res.remoteAddr, pong[]);

assert(client.recv(buf[]) == 4);
assert(buf[] == pong[]);
