    Struct File

    struct File ;

    Constructors

    NameDescription
    this (path, mode) Open a file at path with the options specified in mode.
    this (handle) take ownership of an existing open file handle

    Methods

    NameDescription
    close () close the file
    isOpen () return whether file is open
    move () move operator for file
    read (buf) Read from file into buffer.
    read (bufs) Read from file into multiple buffers. The read will be atomic on Posix platforms.
    seek (offset, whence) Reposition the current read and write offset in the file.
    write (buf) Write buffer content to file.
    write (bufs) Write multiple buffers to file. The writes will be atomic on Posix platforms.

    Example

    auto f = File("temp.txt", mode!"w");
scope (exit)
    remove("temp.txt");
f.write([0, 1]);
