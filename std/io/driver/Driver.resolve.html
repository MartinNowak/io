    Function Driver.resolve

    Resolve hostname using service, family, socktype, and protocol as hints. Calls cb for each resolved AddrInfo. Iteration is terminated early when cb returns a non-zero value.

    abstract int resolve (
      scope const(char[]) hostname,
      scope const(char[]) service,
      AddrFamily family,
      SocketType socktype,
      Protocol protocol,
      scope @nogc @safe int delegate(scope ref const(AddrInfo)) cb
    ) @nogc @safe shared;

    Both hostname and service are already null-terminated for usage with C APIs.

    Returns

    the non-zero value that terminated the iteration or 0 otherwise

    Example

    import std.io.net.addr;
import std.internal.cstring : tempCString;

auto res = driver.resolve("localhost", "http", AddrFamily.IPv4,
        SocketType.stream, Protocol.default_, (ref scope ai) {
            auto addr4 = ai.addr.get!SocketAddrIPv4;
            assert(addr4.ip == IPv4Addr(127, 0, 0, 1));
            assert(addr4.port == 80);
            return 1;
        });
assert(res == 1);
}
