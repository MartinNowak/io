    Interface Driver

    The driver interface used by std.io.

    interface Driver ;

    Swapping this by setting driver or globalDriver allows to exchange the underlying I/O implementation, e.g. for unit-testing or to integrate with an asynchronous event loop.

    Note that switching drivers while Files and Sockets are still open is not @safe and might lead to memory corruption.

    Methods

    NameDescription
    accept (s, addr, addrlen) accept an incoming connection, storing remote addr
    bind (s, addr, addrlen) bind socket to addr
    closeFile (f) create/open file at path in mode
    closeSocket (s) close socket
    connect (s, addr, addrlen) connect socket to addr
    createFile (path, mode) Create/open file at path in mode.
    createSocket (family, type, protocol) create socket
    fileFromHandle (handle) Covert platform-specific file handle to FILE.
    getSocketOption (s, option, opt, optlen) get socket option, type in opt and optlen is SocketOptionType!option
    listen (s, backlog) listen for incoming connections
    localAddr (s, addr, addrlen) get local (bound) addr of socket
    read (f, buf) read from file into buffer
    read (f, bufs) read from file into multiple buffers
    recv (s, buf) read from socket into buffer
    recv (s, bufs) read from socket into multiple buffers
    recvFrom (s, buf, addr, addrlen) read from socket into buffer, storing source addr
    recvFrom (s, bufs, addr, addrlen) read from socket into multiple buffers, storing source addr
    resolve (hostname, service, family, socktype, protocol, cb) Resolve hostname using service, family, socktype, and protocol as hints. Calls cb for each resolved AddrInfo. Iteration is terminated early when cb returns a non-zero value.
    seek (f, offset, whence) seek file to offset
    send (s, buf) send buffer content to socket
    send (s, bufs) send multiple buffer contents to socket
    sendTo (s, buf, addr, addrlen) send buffer content to socket and addr
    sendTo (s, bufs, addr, addrlen) send multiple buffer contents to socket and addr
    setSocketOption (s, option, opt, optlen) set socket option, type in opt and optlen is SocketOptionType!option
    socketFromHandle (handle) Covert platform-specific socket handle to SOCKET.
    write (f, buf) write buffer content to file
    write (f, bufs) write multiple buffer contents to file
